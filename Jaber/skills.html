Quote Sharing Platform

Github

A social media platform designed for users to share, edit, and explore meaningful quotes. The application includes secure Google authentication and is fully deployed on Vercel.

Key features include the ability to post quotes, like quotes shared by other users, and edit previously published posts. The platform also provides search functionality that allows users to find quotes by username, keywords, or tags.

The frontend is built with Vanilla JavaScript for lightweight performance and TailwindCSS for responsive styling. The backend is powered by Node.js and Next.js, which handle server-side logic, API routing, and overall application structure. MongoDB serves as the database for managing user posts and interactions. Deployment is managed through Vercel, ensuring reliable and high-performance hosting. Authentication is implemented through Google OAuth to provide secure and convenient login access.
